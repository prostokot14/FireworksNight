# Fireworks Night
The application for training skills in SpriteKit.
## Description
A game in which you need to click on rockets of the same color to make them explode
## System requirements
* Swift 5.9
* iPadOS 13.0+
## Technologies
* SpriteKit (SKNode, SKSpriteNode, SKLabelNode, SKEmitterNode, SKAction)
* Timer
* UIBezierPath
* Set<UITouch>
